# Basic Commands
 
## 1. `pwd` - Print working directory
```bash
[vagrant@vbox ~]$ pwd
/home/vagrant
```

## 2. `mkdir` - Create a directory/folder in your home directory
```bash
[vagrant@vbox ~]$ mkdir dev
```
## 3. `cd` - Change your current working directory to dev
```bash
[vagrant@vbox ~]$ cd dev/
[vagrant@vbox dev]$
```
